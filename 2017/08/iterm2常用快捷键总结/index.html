iterm2常用快捷键总结

八月 7, 2017 发布在 develop

其实我不太喜欢用tmux, 就跟不喜欢用vim 一样。

iterm2 常用快捷键
新建标签 ⌘ + T
切换标签 ⌘ + 左／右／123
新建水平table ⌘ + D
新建竖直table ⌘ + shift + D
切换table ⌘ + [/]
最大化最小化table ⌘ +shift+↵
关闭标签/关闭table ⌘ + W
在多个terminal间切换 ⌘ +⌥ +123
清除当前行 ctrl + u
清屏1 ctrl + l
行首 ctrl + a
行尾 ctrl + e
自定义跳单词 ⌘ + 前后
删除当前单词 ctrl + w
搜索历史 ctrl + r
上条命令 ctrl + p
删除当前光标后的字母 ctrl + d
删除光标前的字母 ctrl + h
快速填密码 ⌘ + ⌥ + F

其中自定义跳单词设置方法 : itrem2-> preferences -> keys -> 添加如下配置。
